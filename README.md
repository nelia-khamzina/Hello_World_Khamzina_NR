> ## Ниже представлен *мой* код, принимающий имя пользователя и назначенные цифры
``` python
# Принимает на вход имя пользователя и любые цифры для сложения
name = input('Ваше имя:')
a, b = int(input()), int(input())
result = a + b


# Выводит приветствие пользователя и результат сложения
print(f'Приветствую, {name}!')
print(f'Результат сложения {a} и {b} равен {result}')
```
